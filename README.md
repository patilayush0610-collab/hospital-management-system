# Hospital Management System

## 📌 Description
This project is designed to manage hospital operations like patient records, doctor data, appointments, and billing.

## ⚙️ Technologies Used
- Python
- SQL

## 🚀 Features
- Add and update patient records
- Manage doctor information
- Book appointments
- Generate billing

## ▶️ How to Run
1. Download the project
2. Run main.py
